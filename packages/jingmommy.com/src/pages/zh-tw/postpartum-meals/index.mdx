## 坐月子，給自己最好的照顧

生產後的日子，既幸福又充滿挑戰，妳的身體需要足夠的營養來恢復，也需要更多時間來適應新的生活節奏。當寶寶來到這世界，媽媽的身體正在經歷一場蛻變，這時候飲食不只是填飽肚子，更影響恢復速度、體力調整、甚至哺乳順利與否。

### 1. 省時又省心，讓妳專注休息與寶寶

面對產後身體變化、新生兒照顧，時間顯得更加寶貴。我們的月子餐免備料、無需煩惱，簡單加熱即可享用，讓妳在坐月子期間不必為三餐憂心，專注於身體恢復與親子時光。

### 2. 營養均衡，幫助迅速恢復

坐月子是產後黃金修復期，身體的每一個細胞都在自我修復。優質蛋白質、膠原蛋白、溫補食材是恢復的關鍵。我們的月子餐依據產後各階段需求設計，從排毒、修復到滋補，不過度調味、不過度進補，讓補養剛剛好，不造成負擔，幫助一步步回到最佳狀態。

### 3. 美味可口，適合現代媽媽口味

餐點以清爽不油膩為主，融合中西營養理念與烹調方式。多元烹調（如蒸、燉、燜、烤）讓補身同時也吃得開心。每道料理精心設計，保留食材原始營養與美味，滿足不同媽媽需求。

無論是30天完整修復的藥膳餐、精緻餐，15天輕量調理的輕享餐、經濟實惠的元氣餐、幫助快速恢復體態的減醣餐，我們都有適合妳的方案，讓妳隨需求選擇最合適的月子調養法。

產後的食補要採取”階段性食補”的方式，如果不懂方法，一昧的大補特補，不但會補得過多而發胖、導致身材難以恢復，也會影響產後子宮的收縮與惡露的排淨。在這個階段"得宜”是食補的大原則，透過"階段性食補”恢復生理各種機能。把握先排惡露、後補氣血的原則，正確的補養避免發胖，好好的把握住坐月子改變體質的機會。

| 週次 | 重點 |
|------|------|
| 第一週 | 補血、排除惡露、恢復子功機能 |
| 第二週 | 促進代謝、恢復體力、預防產後腰痠背痛 |
| 第三～四週 | 大補氣血、調養體力、抗衰老、增強免疫力 |

### 第一週：平補

<span class="text-amber-900">第一週飲食重點以惡露排淨、加強腎臟排泄功能為目標</span>，以豬肝補血為主。
因為生產後是"虛淤並見”的體質，並不適合大補。所以產後第一週以促進發汗、讓身體利水消腫為目的，將體內多餘的水分排出來，加強新陳代謝。把握第一週"產後利尿期”，才能掌握產後減重的黃金時期!

### 第二週：溫補

<span class="text-amber-900">第二週飲食重點在於預防腰痠背痛、筋骨痠痛、並繼續幫助子宮收縮且將惡露排淨</span>，以麻油腰花為主，佐以杜仲藥膳。利用具有補筋骨的食補來調養產婦因懷孕與生產而鬆弛的筋骨。

### 第三、四週：大補

<span class="text-amber-900">第三、四週大部分的生理機能已獲得大幅度的改善、惡露也將淨</span>，是進補的最佳時機。飲食重點以改善氣虛、增強體力、抗老養顏、補充元氣。這階段以好吃又養身的麻油雞為主，不但可以吃到雞肉甜味、又可喝到濃郁麻油雞湯，越喝身體越暖和。

**完整的30天階段性坐月子調理**不是僅僅進補而已，更是能讓妳**修復損傷，補充體力，強化體質**。讓妳越生越健康、越生越美麗！
