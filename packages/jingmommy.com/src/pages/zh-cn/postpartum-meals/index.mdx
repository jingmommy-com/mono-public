## 坐月子，给自己最好的照顾

生产后的日子，既幸福又充满挑战，妳的身体需要足够的营养来恢复，也需要更多时间来适应新的生活节奏。当宝宝来到这世界，妈妈的身体正在经历一场蜕变，这时候饮食不只是填饱肚子，更影响恢复速度、体力调整、甚至哺乳顺利与否。

### 1. 省时又省心，让妳专注休息与宝宝

面对产后身体变化、新生儿照顾，时间显得更加宝贵。我们的月子餐免备料、无需烦恼，简单加热即可享用，让妳在坐月子期间不必为三餐忧心，专注于身体恢复与亲子时光。

### 2. 营养均衡，帮助迅速恢复

坐月子是产后黄金修复期，身体的每一个细胞都在自我修复。优质蛋白质、胶原蛋白、温补食材是恢复的关键。我们的月子餐依据产后各阶段需求设计，从排毒、修复到滋补，不过度调味、不过度进补，让补养刚刚好，不造成负担，帮助一步步回到最佳状态。

### 3. 美味可口，适合现代妈妈口味

餐点以清爽不油腻为主，融合中西营养理念与烹调方式。多元烹调（如蒸、炖、焖、烤）让补身同时也吃得开心。每道料理精心设计，保留食材原始营养与美味，满足不同妈妈需求。

无论是30天完整修复的药膳餐、精致餐，15天轻量调理的轻享餐、经济实惠的元气餐、帮助快速恢复体态的减糖餐，我们都有适合妳的方案，让妳随需求选择最合适的月子调养法。

产后的食补要采取”阶段性食补”的方式，如果不懂方法，一昧的大补特补，不但会补得过多而发胖、导致身材难以恢复，也会影响产后子宫的收缩与恶露的排净。在这个阶段"得宜”是食补的大原则，透过"阶段性食补”恢复生理各种机能。把握先排恶露、后补气血的原则，正确的补养避免发胖，好好的把握住坐月子改变体质的机会。

| 周次 | 重点 |
|------|------|
| 第一周 | 补血、排除恶露、恢复子功机能 |
| 第二周 | 促进代谢、恢复体力、预防产后腰酸背痛 |
| 第三～四周 | 大补气血、调养体力、抗衰老、增强免疫力 |

### 第一周：平补

<span class="text-amber-900">第一周饮食重点以恶露排净、加强肾脏排泄功能为目标</span>，以猪肝补血为主。
因为生产后是"虚淤并见”的体质，并不适合大补。所以产后第一周以促进发汗、让身体利水消肿为目的，将体内多余的水分排出来，加强新陈代谢。把握第一周"产后利尿期”，才能掌握产后减重的黄金时期!

### 第二周：温补

<span class="text-amber-900">第二周饮食重点在于预防腰酸背痛、筋骨酸痛、并继续帮助子宫收缩且将恶露排净</span>，以麻油腰花为主，佐以杜仲药膳。利用具有补筋骨的食补来调养产妇因怀孕与生产而松弛的筋骨。

### 第三、四周：大补

<span class="text-amber-900">第三、四周大部分的生理机能已获得大幅度的改善、恶露也将净</span>，是进补的最佳时机。饮食重点以改善气虚、增强体力、抗老养颜、补充元气。这阶段以好吃又养身的麻油鸡为主，不但可以吃到鸡肉甜味、又可喝到浓郁麻油鸡汤，越喝身体越暖和。

**完整的30天阶段性坐月子调理**不是仅仅进补而已，更是能让妳**修复损伤，补充体力，强化体质**。让妳越生越健康、越生越美丽！
